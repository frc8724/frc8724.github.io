# mayhem-site

## Developing

### Prerequisites

- [Ruby](https://www.ruby-lang.org/en/downloads) and [Bundler](https://bundler.io/#getting-started)
- [Node.js](https://nodejs.org)

### Setting up

```
bundle install
yarn install
```

### Running

Open two separate terminal windows:

| `bundle exec jekyll serve` | `yarn dev` |
| -------------------------- | ---------- |
