# mayhem-site

## Developing

### Prerequisites

- [Ruby](https://www.ruby-lang.org/en/downloads) and [Bundler](https://bundler.io/#getting-started)
- [Node.js](https://nodejs.org)
- [Yarn](https://classic.yarnpkg.com/en/docs/install)

### Setting up

```
bundle install
yarn install
```

### Running

On Linux/macOS:

`./bin/dev`
